=> 변형 : transition (시간의 제약을 가지고 무언가가 변화를 일으키는 것)
     1) transition-duration : 특정 시간동안 변형효과를 적용하고자 할 때
     2) transition-delay : 특정 시간의 지연효과를 적용하고자 할 때
     3) transition-timing-function : 가속, 감속 효과를 적용하고자 할 때
     4) transition-property : 특정 요소의 변형값을 상이하게 적용하고자 할 때
     => 한번에 모두 사용하고 싶다면?
     => transition: property duration timing-function delay
     => transition: all 0.3s    
     참고 사이트 : cubic bezier

     animation

     animation-duration: 2s;
     animation-iteration-count: infinite;
     animation-timing-function: ease-in;
     animation-direction: alternate
     => 한번에 모두 사용하고 싶다면?
     => animaion: name / duration / function / delay / count / direct

=> 변환 : transform (시간의 제약과는 무관하게 무언가 변화를 일으키는 것)
     : 어떤 요소의 회전값을 적용 : rotate
     : 어떤 요소의 위치값을 적용 : translate
     : 어떤 요소의 사이즈값을 적용 : scale
     : 어떤 요소의 기울기값을 적용 : skew
     => 위 3가지 요소 한번에 같이 사용되는 경우!!
